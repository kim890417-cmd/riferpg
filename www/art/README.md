# 계급 일러스트 넣는 곳

여기에 파일만 넣으면 앱이 자동으로 보여줍니다. 파일이 없으면 기존 이모지로 폴백되므로 깨지지 않습니다.

## 파일 규칙
- 포맷: **WebP** (PNG로 만들었으면 변환 요청하세요)
- 권장 사이즈: 세로 비율 **832×1216** (또는 896×1152) — 앱에서 3:4로 보여줌
- 한 그림이 **큰 진화 연출**과 **계급 옆 작은 원형 아이콘** 둘 다에 재사용됩니다 (위쪽이 잘 보이게 인물 얼굴은 상단~중앙에)

## 파일명 = `<슬러그>-<성별>.webp`  (성별: m=남, f=여)

총 16장 (8계급 × 남/여):

| 슬러그 | 계급 | 남 | 여 |
|--------|------|----|----|
| peasant | 평민  | peasant-m.webp | peasant-f.webp |
| squire  | 종자  | squire-m.webp  | squire-f.webp  |
| knight  | 기사  | knight-m.webp  | knight-f.webp  |
| lord    | 영주  | lord-m.webp    | lord-f.webp    |
| noble   | 귀족  | noble-m.webp   | noble-f.webp   |
| king    | 왕    | king-m.webp    | king-f.webp    |
| emperor | 황제  | emperor-m.webp | emperor-f.webp |
| legend  | 전설  | legend-m.webp  | legend-f.webp  |

> 한쪽 성별만 먼저 만들어도 됩니다. 그 성별로 설정하면 보이고, 없는 칸은 이모지로 폴백.
